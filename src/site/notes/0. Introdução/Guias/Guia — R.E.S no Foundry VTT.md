Esse guia é para usuários do Foundry VTT, ensinando a como configurar para melhor experiência do *Runarcana: Edição Selvagem*.

# Sistema requerido

Vá para o foundry e instale o sistema "[Savage Worlds Adventure Edition](https://foundryvtt.com/packages/swade/)", disponível gratuitamente.

![center|440](/img/user/10.%20Infraestrutura/z_Assets/Images/Prints/print-swade-foundry.png)

# Módulos Principais

- **[Savage Worlds Edição Aventura Regras Básicas](https://foundryvtt.com/packages/swade-core-rules-ptbr) (PAGO)** - Esse modulo não é obrigatório, mas é crucial pelo pacote completo de compêndios contendo o livro de regras, fichas npcs, itens, perícias, vantagens e outros para uso rápido. Você pode comprar a versão em português pela [Retropunk](https://loja.retropunk.com.br/index.php?id_product=591&rewrite=savage-worlds-edicao-aventura-modulo-de-regras-para-foundry-vtt&controller=product).
    
- **[SWADE Fantasy Currencies](https://foundryvtt.com/packages/sfc) (GRATUITO)** - É importante para o universo de runarcana, já que o SWADE por padrão não utiliza mais de uma moeda, e esse modulo vai nos proporcionar adicionar até 4 tipos de moedas. ```O runarcana original, usa 5 tipos de moedas, infelizmente aqui só conseguiremos até 4.```

# Módulos Recomendados

### Gratuitos

- **[Forien's Quest Log](https://foundryvtt.com/packages/forien-quest-log/)** - Quer criar um log de missões com um espaço para fazer anotações detalhadas da história e fazer com que os jogadores sintam-se que estão progredindo, podendo receber recompensas assim que finaliza-las? Forien's é o melhor plugin gratuito para criar suas missões e fazer seus jogadores sentirem que tem um objetivo claro.
    
- **[Mythic's Simpler Quests](https://foundryvtt.com/packages/simpler-quests)** - Similar com o "Forien's Quest Log", o MSQ possui uma ideia similar com a criação de quests, mas com o objetivo de ser bem mais simples, somente focando no objetivo atual e não detalhes ligados a ela. Uma recomendação para que busca algo mais simples.
    
- **[Item Piles](https://foundryvtt.com/packages/item-piles)** - Se você quer criar um baú contendo itens raros, cofres secretos, mercadores que vendem itens para seus aventureiros ou apenas um deposito que seus jogadores possam guardar seus itens, pois já estão pesados por carregar tantos itens inúteis. Item Piles funciona exatamente para tudo e o "SWADE Fantasy Currencies" possui integração com ele, podendo utilizar as novas moedas criadas como moeda de troca dos vendedores.
